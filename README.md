## :memo:기능
+ 유튜브 헤더에 있는 검색 기능을 통해 원하는 비디오 목록을 가져올 수 있다.
+ 검색창 밑으로 비디오 목록을 보여준다.
+ 비디오를 클릭하면 상세 페이지에서 비디오를 시청할 수 있고 우측에 관련 비디오를 볼 수 있다.
+ 반응형 UI
